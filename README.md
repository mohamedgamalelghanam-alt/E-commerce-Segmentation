# 📊 E-commerce Customer Segmentation & Analytics

## 🎯 Project Overview
In this project, I analyzed over **500,000 transactions** from a UK-based retail dataset. I performed **RFM Analysis** and used **K-Means Clustering** to segment customers into distinct groups to help the business target them effectively.

## 🚀 Features
- **Data Cleaning:** Handled missing values and removed returns/negative quantities.
- **RFM Analysis:** Calculated Recency, Frequency, and Monetary metrics for each customer.
- **Machine Learning:** Implemented K-Means clustering to identify 4 customer segments.
- **Interactive Dashboard:** Built a dashboard using **Streamlit** and **Plotly** to visualize sales and segments.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, Scikit-learn, Matplotlib, Seaborn, Plotly
- **Deployment:** Streamlit

## 📈 Key Insights
- Identified **VIP customers** who contribute to the majority of revenue.
- Detected **at-risk customers** who haven't purchased in months.
- Visualized sales trends to find the peak selling months.

## 🏃 How to run the Dashboard
1. Install requirements: `pip install -r requirements.txt`
2. Run app: `streamlit run app.py`
